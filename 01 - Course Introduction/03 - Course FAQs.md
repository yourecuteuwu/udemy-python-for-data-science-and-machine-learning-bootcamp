# Course FAQs
1. **Do I need to use Juypter Notebook and Anaconda?**

Sol. Nope. All of the code is compatible with any IDE that supports Python. You can even convert the notebook into .py file.

2. **Where do I get the notebooks for this course?**

Sol. Can get them through two ways: as the zipped up files in the lectures or in the lecture titled: "Juypter Notebooks"

3. **Is there a good companion book for this course?**

Sol. There is, it is called ISLR, an Introduction to Statistical Learning

7. **How do I see the Docstring and the method pop ups in Juypter Notenook?**

Sol. For method pop ups, use Tab. For example with `my_list = [1, 2, 3]`, when you do this `my_list.` and click Tab, you will get a list of methods. For Docstring, you need to perform Shift + Tab.

9. How do I know where my notebooks are being saved?

Sol. If you run the command `pwd` in your notebook, you will see the directory for which your notebooks are being saved.

10. How do I change where the notebooks are being saved?

Sol. To do that, you must `cd` into a different directory.

11. How to open `.ipynb` files

Sol.
- You first need to install Python and Juypter Notebook. There is a guide on Python for that.
- Put your `.ipynb` into some directory.
- Navigate to that directory in terminal using `cd`
- Type `juypter notebook` and a web browser should pop up.
- Click on the notebook you want to see in the browser
- Done!
