# Introduction to the Course

Will cover Python, NumPy, SciPy, Pandans, SciKit, Matplot, Plotly and more

- Learn **Python**, the fundmentals
- Learn **NumPy** to work with data arrays
- Learn **Pandas**, to analyze data and read it from multiple docs like csv, txt, etc.
- Learn **Matplotlib** (basic data visualization)
- Learn **Seaborn** (complex data visualization)
- Learn about **SciKit** Library for implementing machine learning algorithms like regression, decision trees, etc.
- Learn Big Data with **AWS**
- Learn **TensorFlow**, for deep learning
